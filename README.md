# Intro
This is my humble attempt at the final project for the Odin Project's Foundations track. [link](https://www.theodinproject.com/lessons/foundations-calculator)

I had an idea that I wanted this to feel nautical/day at the beach so I did my best to select colors that reflected this palette. 

## Features
- Basic calculator operators (+, -, /, *)
- Advanced operators (%, decimals, plus/minus)
- Snarky text 
- Limited keyboard support for entering numbers

## Existing bugs
- Returning calculations with "Enter" sets the result to "0"
